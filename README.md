# 🎧 Spotify Backend – Music Streaming API

A scalable backend system for a music streaming platform inspired by Spotify.  
This project focuses on building a **secure and modular REST API architecture** capable of handling user authentication, playlist management, and music catalog operations.

The backend is designed with **Node.js and Express**, follows a **modular MVC-style structure**, and implements **JWT-based authentication**, scalable database models, and RESTful API principles.

---

## 🚀 Tech Stack

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB (Mongoose ODM)

**Authentication & Security**
- JWT (JSON Web Tokens)
- bcrypt (Password Hashing)

---

## 📌 Features

- Secure **User Authentication & Authorization**
- **JWT-based login system**
- Password hashing using **bcrypt**
- **Playlist creation and management**
- Music catalog management
- RESTful API design
- Modular backend architecture
- Middleware-based authentication
- Centralized error handling
- Scalable database schema using MongoDB

---

## 🏗️ Backend Architecture

The project follows a **modular backend architecture** to ensure maintainability and scalability.
